จุดประสงค์
	Event-line สร้างขึ้นเพื่อให้สามารถเพิ่มอีเว้นท์ภายในเกมได้โดยง่ายต่อผู้พัฒนา

ประเภทของ Event-line
	Main Event-line เป็นอีเว้นท์หลักโดยจำเป็นจะต้องเกิดอีเว้นท์ที่อยู่บรรทัดข้างบนก่อนเสมอ
	Side Event-line เป็นอีเว้นท์ย่อย ลำดับการเกิดไม่อิงบรรทัดแต่จำเป็นต้องอิงที่มา

ตัวอย่าง 1 Main Event-line
	ชายชรามอบอาวุธให้กับเด็กสาวปริศนา
	เด็กสาวปริศนาฆ่าชายชราคนนั้น
	เด็กสาวเดินจากไปอย่างเงียบๆ

ตัวอย่าง 2 Side Event-line
	ถูกกิลด์มาสเตอร์มอบหมายให้ไปหาแอปเปิ้ลทองคำ ( ME-เด็กสาวเดินจากไปอย่างเงียบๆ )
	ปีศาจบุกอย่างกระทันหันที่เมืองฟอริน่า ( SE-ถูกกิลด์มาสเตอร์มอบหมายให้ไปหาแอปเปิ้ลทองคำ )
